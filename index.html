Important: Backup Your Configuration

Before proceeding with the firmware update, we strongly recommend creating a backup of your existing configuration.

How to Save a Backup to a USB Stick:

  1. Insert a USB stick into the OceanBridge USB port.
  2. Navigate to: Menu > System > Backup & Restore.
  3. Select 'SAVE BACKUP'.
  4. Add a name and choose USB as the location.
  5. Select 'SAVE' to store the backup.
